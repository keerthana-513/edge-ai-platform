# Edge AI Inference Monitoring Platform

A lightweight, real-time monitoring platform designed for Edge AI applications to track and analyze inference events from edge devices.

## Features
- **REST API:** Easily log AI inference data (device ID, model ID, result) via a secure endpoint.
- **Real-time Dashboard:** A modern, auto-refreshing web interface to visualize incoming logs.
- **Analytics:** Built-in metrics cards displaying "Total Detections" and "Recent Activity" (last 1 hour).
- **Lightweight:** Designed to run efficiently on low-resource hardware like Raspberry Pi.
